# Download from Build-A-Table export

Attempts to download data via VDOE's Build-A-Table tool. This is a
fallback method that may require browser automation.

## Usage

``` r
download_buildatable_export(end_year, level)
```

## Arguments

- end_year:

  School year end

- level:

  "school" or "division"

## Value

Data frame with enrollment data
